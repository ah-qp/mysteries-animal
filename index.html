엉덩이로 숨쉬는

동양비단거북

( Eastern painted turtle )

거북이는 외부온도에 따라 변하는
변온 동물로 추운 겨울 자신의 체온을
떨어뜨려 신진대사 속도를 늦춘다.
이 때 동양비단거북이는 최소한의
호흡을 하기 위해 엉덩이를 이용한다.

변을
먹는다.

:

변을 먹는

토끼

( Rabbit )

토끼는 1차 소화과정에서 음식물의
영양분을 모두 흡수하지 못한다.
특히 섬유질을 모두 흡수하지 못해
토끼는 자신이 싼 변 중에서 묽은 변을
섭취해 영양분을 완벽하게 섭취하려 한다.

영원히
산다.

:

영원히 사는

작은보호탑해파리

( Turritopsis nutricula )

해파리는 번식 후에 영유아 상태인 폴립상태로 돌아가 다시 성장한다. 외부의 환경이
변하거나 다른 생물에 잡아먹히지 않는 한 성장을 되풀이하며 영원히 살아간다.

서서
잔다.

:

서서 자는

말

( Horse )

말은 야생에서 맹수의 공격을 피해 언제든 도망갈 수 있게 서서 자는 본능이 발달했다. 만약에 말이 안전함을 느낀다면 누워서 잔다. 보통은 하루에 30분 정도 누워서 잔다.

배변을
안한다.

:

배변을 안하는

곰

( Bear )

곰은 겨울잠을 자는 동안에 배뇨 배변을
하지 않는다. 신장의 벽을 통해 자기 배설물을 흡수해서 주요 성분인 질소화합물을
자기 영양분으로 흡수할 수 있기 때문이다. 자기의 배설물을 자면서 재활용하고 있다.

최초로
재판을
받았다.

:

최초로 재판을 받은

벌

( Bee )

벌이 저지른 살인죄로 864년 최초로
동물 재판이 독일에서 열렸다. 벌은 사람을 침으로 쏘아 죽인 죄로 재판에 회부되었고 사형 판결을 받았다. 벌의 둥지를 막아
질식사 시키는 것으로 사형을 집행하였다.

날개 없이
날아다닌다.

:

날개 없이 날아다니는

크리코펠리아 파라디시

( Chrysopelea paradise )

뱀은 활공을 하여 나무에서 나무로
날아서 이동한다. 더 멀리 날기 위해
가지에서 뛰어오르기도 한다. 이 때
몸을 납작하게 만들어 S자 모양으로
미끄러지듯 안정된 활강을 한다.

활공
엔진에 의한 추진력 없이 무동력상태로 강하하는 비행

매운맛을
느낄 수 없다.

:

매운맛을 느끼지 않는

하이펠트 두더지쥐

( Highveld mole rat )

쥐는 대부분 굴에서 AITC의 성질이 있는
독개미의 공격을 받는다. 덕분에 이들 쥐는 AITC 내성이 생겨 다른 두더지쥐들이
접근하기 꺼리는 곳에서도 살 수 있다.

AITC
세포 내 단백질을 적극적으로 손상하는 화학 화합물
매운 것을 먹을 때 고통을 느끼게 만듦

강에서
산다.

:

강에서 사는

강돌고래

( River dolphin )

강돌고래는 몸집이 작고 물고기를 직접
사냥해 먹을 수 있는 긴 부리와 이빨을
가지고 있다. 아마존강, 갠지스강, 양쯔강
매콩강, 인더스강에서 살고 있는데 환경오염으로 이미 멸종이거나 점차 사라지고 있다.

물을
안 먹어도
괜찮다.

:

물을 안 안먹는

캥거루쥐

( Kangaroo rats )

쥐는 건조한 지역에 살지만 먹이로부터
수분을 얻을 뿐 거의 물을 마시지 않는다.
또한 콩팥에서 수분이 재 흡수 되어 매우
진한 오줌을 소량 배설한다.

유배를
갔다.

:

유배를 간

코끼리

( elepant )

조선시대 태종 임금은 일본에서 코끼리를
선물로 받았다. 이듬해 이우라는 사람이
코끼리를 보고 '이상하게 생겼다.'며 침을
뱉고 비웃다가 화가 난 코끼리에게 밟혀
죽었다. 이 코끼리는 재판을 받아 지금의
소록도인 노루섬으로 유배를 갔다.

심박수를
조절한다.

:

심박수를 조절하는

대왕고래

( Blue Whale )

대왕고래의 대동맥 활은 탄력이 강해 혈류를 조절하는 기능이 탁월하다. 상황에 맞게
혈류를 조절해서 1분에 2회에서 30여 회까지 심박수를 변화시킨다.
